# Product Requirements Document (PRD)

## Project Title
ППП: Генератор документів на основі шаблонів (Next.js + TypeScript)

## Опис проекту
Веб-додаток (Next.js + TypeScript) для генерації документів (docx) на основі шаблонів і введених користувачем даних. Дані зберігаються через Prisma ORM у базі даних Postgres. Генерація документів здійснюється через зовнішній REST API, який приймає шаблон (у форматі base64 docx) та параметри (JSON), і повертає заповнений документ (base64 docx).

## Основні функції
- Аутентифікація користувача (логін/лог-аут, MVP: admin:password у БД)
- Дашборд після входу з меню навігації праворуч
- Меню навігації (сторінки):
    - Генерація документів (основна функція)
    - Посадові особи
    - Шаблони
- Інтерфейс українською мовою

## Сторінки та функціонал

### 1. Сторінка генерації документів
- Меню навігації праворуч
- Дропдаун для вибору шаблону
- Після вибору шаблону — форма з необхідними полями
- Дропдаун для вибору посадової особи (її дані підставляються у форму)
- Кнопка для генерації документа
- Виклик зовнішнього API:
    - URL: https://n1mrvi9bc7.execute-api.eu-west-1.amazonaws.com
    - Метод: POST
    - Тіло:
      ```json
      {
        "params": { /* поля з форми */ },
        "template": "base64 docx"
      }
      ```
    - Відповідь: base64 docx (заповнений документ)

### 2. Сторінка посадових осіб
- Меню навігації праворуч
- Список посадових осіб
- CRUD-операції (створення, редагування, видалення)
- Поля посадової особи:
    - Ім'я
    - Прізвище
    - По батькові
    - Посада (довгий рядок)
    - Звання
    - Паспортні дані
    - Дата з
    - Дата по
    - Адреса (довгий рядок)

### 3. Сторінка шаблонів
- Меню навігації праворуч
- Список шаблонів документів
- Назва шаблону
- Список властивостей, які потрібно заповнити (всі поля — довгі рядки)
- CRUD-операції для шаблонів
- Для кожного шаблону прикріплений docx-файл:
    - Можна завантажити docx-файл шаблону
    - Можна замінити docx-файл шаблону
    - docx-файл зберігається у базі даних

## Технології
- Next.js (TypeScript)
- Prisma ORM
- Postgres
- REST API для генерації документів (docx, base64)

## MVP
- Простий логін (admin:password у БД)
- Генерація документів через API
- CRUD для посадових осіб і шаблонів
- Український інтерфейс

## Додатково
- Всі сторінки мають однакове меню навігації праворуч
- Валідація форм
- Зберігання шаблонів і посадових осіб у БД
- Вивід помилок користувачу
